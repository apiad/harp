# Harp

[![Version](https://img.shields.io/badge/version-v0.1.0-blue.svg)](https://github.com/apiad/harp/releases/tag/v0.1.0)

A background daemon for Wayland that transcribes audio and types results.

## Overview
Harp is a Linux background daemon that listens for a global hotkey, records audio, transcribes it via OpenRouter's OpenAI-compatible Audio API, and types the result into the currently active Wayland window.

## Status
Currently in initial development.
